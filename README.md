# h0stream

**h0stream** is a clean, minimalist example implementation of a streaming service’s frontend paired with backend-like functionality. It provides a lightweight demonstration of how basic UI components and flows—such as listings, media playback, and detail pages—can be structured using web technologies.

---

##  Features

- **Frontend-focused prototype** using pure HTML, CSS, and JavaScript.
- **Multiple views** including:
  - A series listing (index) page.
  - A detail page for each series.
  - A media player interface for simulating playback.
  - A mock account page showcasing user-style layout.
- **Modular file structure**:
  - `index.html`
  - `series-detail.html`
  - `player.html`
  - `account.html`
  - `assets/` directory for shared styles, scripts, or images.
